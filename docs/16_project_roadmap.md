# 16. Project Roadmap

## Phase 1 — MVP Analytics

Dataset, Streamlit overview, filters, KPIs, tread-depth analysis, geo map, README and HTML.

## Phase 2 — Analytics Engineering

SQLite, DuckDB, dbt staging/intermediate/marts, tests and docs.

## Phase 3 — Statistical & Clustering Layer

Automatic tests, correlations, K-Means, DBSCAN, PCA/UMAP and interpretation.

## Phase 4 — Machine Learning

Predict wear rate, remaining life and risk class.

## Phase 5 — Deep Learning

TensorFlow, PyTorch, CNN for images and embeddings for recommendation.

## Phase 6 — Productization

API, Docker, CI/CD, MLflow, deployment and monitoring.
