# 01. Project Context

TyreWear Intelligence is a portfolio project for tyre analytics. It studies tread depth, wear rate, remaining life, safety, energy efficiency, noise, price and geography.

The business goal is to avoid choosing tyres only by price. The analytical goal is to combine descriptive analytics, statistics, clustering, machine learning and recommendation logic.

Main questions:

- Which tyres last longer?
- Which brands have lower wear rate?
- Which regions have higher risk?
- Can we predict remaining life?
- Can we recommend tyres for a user profile?
