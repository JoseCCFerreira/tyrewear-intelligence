# 02. Role-Based Breakdown

## Data Analyst

KPIs, filters, dashboards, maps, comparisons and business storytelling.

## Analytics Engineer

SQLite, DuckDB, dbt staging, intermediate models, marts, tests and documentation.

## Data Scientist

Hypothesis tests, correlation, regression, clustering and model interpretation.

## Data Engineer

Data ingestion, validation, raw/processed/output layers and reproducible pipelines.

## ML Engineer

Training scripts, model comparison, deep learning examples, inference and recommendation integration.
