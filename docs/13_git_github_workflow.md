# 13. Git and GitHub Workflow

Create repository:

```bash
git init
git add .
git commit -m "Initial project structure"
git branch -M main
git remote add origin <repo-url>
git push -u origin main
```

Suggested branches:

- main
- develop
- feature/streamlit-dashboard
- feature/dbt-models
- feature/statistical-tests
- feature/ml-models
- feature/geo-map
- feature/deep-learning
- docs/html-landing-page

Use issues, milestones, releases and GitHub Projects to manage the roadmap.
