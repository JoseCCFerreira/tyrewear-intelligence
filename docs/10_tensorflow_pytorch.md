# 10. TensorFlow and PyTorch

TensorFlow and PyTorch should be used with purpose.

Good use cases:

- CNN for tyre images: normal, worn, bald, cracked, irregular wear, defective
- neural networks for tabular regression/classification as a comparison layer
- LSTM/GRU/Temporal CNN for repeated TD time series
- embeddings for recommendation engines

For small tabular datasets, Random Forest or Gradient Boosting may outperform neural networks.
