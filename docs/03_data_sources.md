# 03. Data Sources

The MVP uses a realistic simulated dataset. This is appropriate for portfolio development because it allows a complete pipeline without licensing issues.

Future sources:

- NHTSA / UTQG
- EPREL
- allowed tyre review datasets
- weather APIs
- geographic datasets
- owned tread-depth measurements
- tyre images

The generated dataset is stored at `data/raw/sample_tyre_dataset.csv`.
