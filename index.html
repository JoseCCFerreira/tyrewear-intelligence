The Problem

Buying tyres by price alone hides the real cost.

A cheaper tyre can become more expensive when it wears faster, increases energy consumption, performs worse in wet conditions, or creates higher replacement risk. Price is only one variable in a decision that also depends on mileage, tread depth, safety class, driving context and long-term operating cost.

TyreWear Intelligence reframes tyre choice as an analytical decision: compare performance, quantify uncertainty and surface the trade-offs that matter before purchase or fleet replacement.

The Solution

A local-first platform for tyre intelligence.

The project combines structured storage, analytical modelling, statistical testing, machine learning and an interactive advisor into one reproducible workflow.

How to run

Local execution workflow.

The project is designed to run locally with a clear path from ingestion to dashboard. 

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

python scripts/load_sqlite.py
dbt build
python scripts/train_models.py
streamlit run app.py
