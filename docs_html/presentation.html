Project presentation

From tyre data to a defensible decision.

TyreWear Intelligence explains, compares and recommends tyres using tread depth, wear rate, cost, safety, geography, statistics, clustering and machine learning.

Objective

The project helps a user move beyond price-only tyre selection. It turns tyre measurements and product attributes into evidence about durability, risk, cost per kilometre and expected remaining life.

Understand

Explore tread depth, wear, cost, risk and dimension patterns.

Compare

Rank brands, tyre sizes, regions and candidate tyres under the same rules.

Decide

Use a weighted decision score and export the analysis for review.

Process

Reference data Raw panel Cleaning SQLite DuckDB dbt marts Statistics ML Decision

The pipeline creates a realistic measurement dataset, adds an official NHTSA UTQGS reference layer, stores data locally, builds analytics marts and exposes the results through Streamlit and static HTML documentation.

What The App Does

Analysis

KPIs, distributions, tread-depth views, geographic maps, dimension heatmaps and monthly patterns.

Evidence

Hypothesis tests, p-values, correlations, clustering metrics and model comparison.

Recommendation

Weighted scoring by durability, safety, cost, comfort, efficiency and risk.

Decision Center

The Decision Center lets the user choose country, vehicle type, tyre size, budget and business priorities. The app recalculates a decision score and explains the best candidate with supporting metrics.

  • Durability: remaining useful life and wear behaviour.
  • Safety: wet grip and risk class.
  • Economics: price and cost per 1000 km.
  • Comfort and efficiency: noise, fuel efficiency and rolling resistance.

Exports

The analysis can be exported directly from Streamlit:

CSV

For Excel, BI tools or additional modelling.

JSON

For reproducible decision records and integration.

HTML Report

For sharing the recommendation and top candidates with non-technical stakeholders.

Important Data Note

The NHTSA UTQGS layer is official real reference data. The repeated tread-depth-by-mileage panel is realistic simulated data because public tyre-level wear measurements by mileage, geography and usage profile are not generally open.
